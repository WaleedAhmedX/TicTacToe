# Java Swing Tic-Tac-Toe Game

A desktop-based Tic-Tac-Toe game built entirely in Java using the Swing API for the graphical interface. I developed this project during my second semester to apply core Object-Oriented Programming (OOP) principles in a practical, visual application.

## 🚀 Features & Components
* **User Authentication:** Includes a `LoginApp` window that serves as the entry point before the game starts.
* **Interactive UI:** Built using standard Swing components (`JFrame`, `JButton`, layouts) to manage the game grid.
* **Visual Additions:** Features a custom loading or status animation via `BeautifulProgressBarFrame`.
* **Audio Integration:** Uses `music.wav` for background sound effects and event handling during gameplay.

## 🛠️ Concepts Applied
* **Encapsulation:** Keeping game state, player turns, and win-checking logic protected inside specific classes.
* **Inheritance:** Extending standard Java Swing classes to create customized frames and interface behaviors.
* **Event Handling:** Implementing ActionListeners to handle user mouse clicks and trigger UI updates smoothly.
